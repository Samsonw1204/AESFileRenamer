# AESFileRenamer

A small Python tool for batch-renaming AES skill evaluation videos.

Standard rename sequence:

1. Lifeguard Introductions.MP4
2. Skill Evaluation 1--Attempt 1.MP4
3. Skill Evaluation 1--Attempt 2.MP4
4. Skill Evaluation 2--Attempt 1.MP4
5. Skill Evaluation 2--Attempt 2.MP4
6. Skill Evaluation 3--Attempt 1.MP4
7. Skill Evaluation 3--Attempt 2.MP4
